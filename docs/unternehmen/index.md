# 🏢 Unternehmen

Ein Unternehmen ist jede Art von Organisation — eine Firma, ein Verein, eine Behörde, eine Praxis. Hier verwaltest du alles, was du über eine Organisation wissen musst.

---

## Was ein Unternehmen ausmacht

### Stammdaten

| Feld | Beschreibung |
|------|-------------|
| **Name** | Pflichtfeld — der gebräuchliche Name |
| **Handelsname** | "Doing Business As" — der Name, unter dem die Firma auftritt |
| **Offizieller Name** | Der im Handelsregister eingetragene Name |
| **Rechtsform** | GmbH, AG, UG, GbR, Einzelunternehmen, etc. |
| **Branche** | In welchem Sektor die Organisation tätig ist |
| **Registernummer** | Handelsregisternummer |
| **Steuernummer** | Steuerliche Identifikation |
| **USt-IdNr.** | Umsatzsteuer-Identifikationsnummer |
| **Website** | URL der Unternehmenswebsite |
| **Land** | Hauptsitz |
| **Beschreibung** | Ausführliche Beschreibung der Organisation |
| **Notizen** | Interne Anmerkungen |

> 💡 **Anzeige-Logik:** Das CRM zeigt automatisch den passendsten Namen: Handelsname vor offiziellem Namen vor dem Standard-Namen. So siehst du immer den gebräuchlichsten Namen.

### Kontaktdaten

Wie bei Kontakten — mehrere Einträge pro Kategorie möglich:

- 📧 **E-Mail-Adressen** — z. B. info@firma.de, buchhaltung@firma.de
- 📱 **Telefonnummern** — Zentrale, Fax, Hotline, etc.
- 📮 **Postadressen** — Hauptsitz, Niederlassung, Rechnungsadresse, etc.

---

## Unternehmen anlegen

Klicke in der Unternehmensliste auf **+ Unternehmen**:

1. **Name** (Pflicht)
2. **Status** (Pflicht)
3. Optional: E-Mail, Telefon, weitere Details

Alle weiteren Felder pflegst du direkt auf der Detail-Seite.

---

## Unternehmen finden

### 🔍 Suche

Die Volltextsuche durchsucht:
- Name, Handelsname, offizieller Name
- Website
- Beschreibung und Notizen
- E-Mail-Adressen und Telefonnummern

### 🔽 Filter

| Filter | Was er tut |
|--------|-----------|
| **Status** | Kunde, Partner, Lieferant, etc. |
| **Branche** | Nach Branchenzugehörigkeit |
| **Rechtsform** | GmbH, AG, etc. |
| **Land** | Nach Hauptsitz-Land |
| **Erstellt am** | Zeitraum der Erstellung |

---

## Verknüpfte Kontakte

Das Spannende an Unternehmen: die **Personen dahinter**. Auf der Unternehmensseite siehst du alle verknüpften Kontakte — mit Rolle, Position und Zeitraum.

Du kannst:
- Bestehende Kontakte dem Unternehmen zuordnen
- Die Rolle definieren (Geschäftsführer, Angestellter, Berater, etc.)
- Einen **Hauptansprechpartner** markieren
- Zeiträume setzen (seit wann, bis wann)

Mehr dazu unter [Beziehungen](../beziehungen).

---

## Massenoperationen

Wie bei Kontakten: Mehrere Unternehmen auswählen und gemeinsam Status ändern oder löschen.

---

## Aktivitäten

Jede Änderung wird protokolliert. Die Timeline auf der Unternehmensseite zeigt, wer wann was geändert hat — und du kannst manuelle Notizen hinzufügen.
